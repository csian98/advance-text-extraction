## Installation
``` sh
git clone git@github.com:csian98/advance-text-extraction.git
cd advance-text-extraction
uv init
uv add -r requirements.txt
```

## Execution

### PDF Data Extraction
``` sh
uv run pdf_main.py
```

### Web Scraping & SnowFlake Data upload
``` sh
uv run web_main.py
```
